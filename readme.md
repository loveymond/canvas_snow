Readme
======

用Canvas做成的雪景。

调用方法：

```javascript

    /**
     * @param canvasId, String, canvas元素的ID
     * @param speed, Float, optional, 雪花下落速度，0.01-100，默认值0.5
     * @param density, Integer, optional, 雪花的密度，数值越小密度越大，最小为1，默认100
     * @param snowColor, String, optional, 雪花的颜色，同canvas颜色，rgba用法与CSS不同，需要注意。默认#fff
     */    
    snow(canvasId, speed, density, snowColor)

```
